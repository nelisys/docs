# moment

## format

format()

```jacascript
moment().format('YYYY-MM-DD HH:MM:ss');
```

## formats

| format     | example    |
| ---------- | ---------- |
| YYYY-MM-DD | 2019-08-07 |
| D MMM YYYY | 7 Aug 2019 |

## convert

convert from unix timestamp

```javascript
moment.unix(timestamp);
```

## human

```javascript
moment().toNow(true);
```
