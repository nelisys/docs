# set time zone

```console
$ sudo timedatectl set-timezone Asia/Bangkok
```

```console
$ timedatectl
                      Local time: Sat 2019-08-18 07:20:49 +07
                  Universal time: Sat 2019-08-18 00:20:49 UTC
                        RTC time: Sat 2019-08-18 00:20:50
                       Time zone: Asia/Bangkok (+07, +0700)
       System clock synchronized: no
systemd-timesyncd.service active: yes
                 RTC in local TZ: no
```

```console
$ date
Mon Aug 19 07:21:25 +07 2019
```
