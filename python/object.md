# object

## get keys

```py
my_dict = {'key1': 10, 'key2': 20, 'key3': 30}

# Loop through keys
for key in my_dict:
    print(key)

# key1
# key2
# key3
```

## foreach

```py
my_dict = {'key1': 10, 'key2': 20, 'key3': 30}

# Loop through keys and values
for key, value in my_dict.items():
    print(f'Key: {key}, Value: {value}')

# Key: key1, Value: 10
# Key: key2, Value: 20
# Key: key3, Value: 30
```
