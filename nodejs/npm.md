save it as a dependency in the package.json file.

Press ^C at any time to quit.

## init -y

or run with option '-y'

```console
$ npm init -y
Wrote to /Users/supasin/test/package.json:

{
  "name": "test",
  "version": "1.0.0",
  "description": "",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": [],
  "author": "",
  "license": "ISC"
}
```
