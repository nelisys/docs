# GY-906 Infrared Temperature Sensor Module

## Wiring

```
 /---------\     /-----------\
 | GY-906  |     | Pi        |
 +---------+     +-----------+
 | VIN     |-----| 01 3.3v   |
 | GND     |-----| 06 Ground |
 | SCL     |-----| 05 SCL    |
 | SDA     |-----| 03 SDA    |
 \---------/     \-----------/
```
