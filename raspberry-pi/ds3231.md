# DS3231 Real Time Clock (RTC)

## Wiring

```
 /---------\     /-----------\
 | DS3231  |     | Pi        |
 +---------+     +-----------+
 | SCL     |-----| 05 SCL    |
 | SDA     |-----| 03 SDA    |
 | VCC     |-----| 01 3.3v   |
 | GND     |-----| 06 Ground |
 \---------/     \-----------/
```
